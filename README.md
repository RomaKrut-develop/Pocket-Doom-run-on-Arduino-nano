# Run the Classic Doom Game with SSD1306 and Arduino nano

Witness the magic where the legendary Doom meets the compact power of Arduino nano and the versatile SSD1306 display.

We removed the sounds in our code because we wanted to reduce the weight of the code and remove the buzzer, which takes up a lot of space and sounds unpleasant.

link to the project:

## Connecting equipment
**Display connection:**





GND to GND on the board



VCC to 3.3V on the board



SDA to A4 on the board



SCL (SCK) to A5 on the board

**Button connection:**





Connect GND from the board to all buttons



Connect the "Up" button to port 8 on the board



The "Down" button to port 3 on the board



The "Left" button to port 6 on the board



The "Right" button to port 7 on the board



The "Shot" button to port 10 on the board

## Game Map
To make this project possible, I had to make some compromises, allowing us to see a functioning prototype of Doom running on one of the smallest Arduino products available. The current version is operating with 90% of the available memory, meaning that we are rapidly approaching the limit of what is achievable. One adjustment I made was to include only one level (based on E1M1 from Wolfenstein 3D). So, when you clear a level and advance to the next, you are essentially replaying the same level repeatedly.
